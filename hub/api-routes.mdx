title: "API routes"
description: "All FastAPI routes, authentication requirements, query parameters, and response shapes."
---

All routes require a `Bearer {WorkOS JWT}` in the `Authorization` header unless otherwise noted.

<Note>
  The following routes do **not** require authentication: `GET /api/health`, `POST /api/admin/sync/*` (EventBridge), and `GET /api/admin/zoominfo/callback` (Okta redirect).
</Note>

## Accounts

| Method | Path | Description |
|---|---|---|
| `GET` | `/api/accounts` | List with filters, search, sort, and pagination |
| `GET` | `/api/accounts/filter-options` | Distinct stages and owner names |
| `GET` | `/api/accounts/{id}` | Full account detail with profile, health, and counts |
| `POST` | `/api/accounts/{id}/refresh-intel` | Enqueue `company_intel` task |
| `POST` | `/api/accounts/{id}/check-calls` | Enqueue `health_score` task |
| `POST` | `/api/accounts/{id}/regenerate-loss-analysis` | Enqueue `loss_analysis` task |
| `GET` | `/api/accounts/{id}/health` | Latest health score with full breakdown |
| `GET` | `/api/accounts/{id}/health/history` | Health trend data (default 90 days) |
| `GET` | `/api/accounts/{id}/loss-analysis` | Loss analysis for the account |
| `PATCH` | `/api/accounts/{id}/loss-analysis` | Override category and notes |
| `GET` | `/api/accounts/{id}/calls` | Paginated call summaries (`call_date DESC`) |
| `GET` | `/api/accounts/{id}/actions` | Action items for the account |
| `POST` | `/api/accounts/{id}/meeting-prep` | Request a meeting prep brief |
| `GET` | `/api/accounts/{id}/meeting-prep` | List past briefs for the account |
| `GET` | `/api/accounts/{id}/meeting-prep/{brief_id}` | Full brief detail |
| `GET` | `/api/accounts/{id}/zoominfo/suggest-titles` | Rules-based title suggestions |
| `POST` | `/api/accounts/{id}/zoominfo/suggest-titles-ai` | AI-powered title suggestions |
| `POST` | `/api/accounts/{id}/zoominfo/search` | Search ZoomInfo contacts |
| `POST` | `/api/accounts/{id}/zoominfo/enrich` | Enrich and save contacts |
| `GET` | `/api/accounts/{id}/contacts` | List saved ZoomInfo contacts |
| `GET` | `/api/accounts/{id}/zoominfo/fetch-log` | ZoomInfo fetch history |

### Query parameters for `GET /api/accounts`

| Param | Type | Default |
|---|---|---|
| `page` | int ≥1 | 1 |
| `per_page` | int 1–100 | 50 |
| `status` | string | — |
| `stage` | string | — |
| `owner_email` | string | — |
| `search` | string | — (name or domain ILIKE) |
| `sort_by` | `name` \| `deal_size` \| `health_score` \| `last_call_date` \| `owner_name` \| `updated_at` | `updated_at` |
| `sort_dir` | `asc` \| `desc` | `desc` |

## Admin
