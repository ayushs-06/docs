---
title: "Database schema"
description: "PostgreSQL schema for all Hub tables, managed by Alembic."
---

The Hub uses **PostgreSQL 16**. All tables use UUID primary keys. Alembic manages migrations via an async `env.py`. Migrations run automatically on Docker startup.

## Base model

All models extend `Base`, which provides:

- `id: UUID` — primary key, auto-generated
- `created_at: DateTime(timezone=True)` — server default `now()`

Computed daily from Snowflake `DAILY_ENTERPRISE_*` tables. No AI is involved. Currently disabled (`HEALTH_SCORES_ENABLED=false`).

| Column | Type | Notes |
|---|---|---|
| `id` | UUID PK | |
| `account_id` | UUID FK → `accounts` CASCADE | |
| `computed_date` | DATE NOT NULL | |
| `overall_score` | FLOAT NOT NULL | weighted composite 0–100 |
| `overall_status` | VARCHAR(10) | `green` (≥70) \| `yellow` (≥40) \| `red` (&lt;40) |
| `priority_score` | FLOAT | from `DAILY_ENTERPRISE_PRIORITY_SCORE` |
| `adoption_score` | FLOAT | from `DAILY_ENTERPRISE_ADOPTION_LEVEL_SCORE` |
| `relationship_risk_score` | FLOAT | from `DAILY_ENTERPRISE_RELATIONSHIP_RISK_SCORE` |
| `days_since_last_meeting` | INTEGER | |
| `crash_count` | INTEGER | from `MONTHLY_ENTERPRISE_CRASHES` |
| `weights_used` | JSONB | for auditability |
| `created_at` | TIMESTAMPTZ | inherited |

**Unique constraint:** `(account_id, computed_date)`
